## Tp Utils

提供一些额外传送命令的 [TShock](https://github.com/Pryaxis/TShock) 插件。

### 所有可用命令

- `tpd`

    传送到上次死亡的位置。

    可能的结果：
    - 成功：`已传送到：{X}, {Y}`
    - 失败：`还没有记录过死亡位置`

- `add <name>`

    将当前位置添加到收藏位置列表中，name 仅用于备注。

    可能的结果：
    - 成功：`已收藏：<{id}> ({X}, {Y}) {name}`
    - 失败：`输入格式错误，用法：add <name>`

- `ls`

    列出所有已收藏的位置。

    可能的结果：
    - 成功：
        ```
        所有已标记的位置：
          <{id}> ({X}, {Y}) {name}
        共计：{Count} 个标记
        ```

- `tpl <id>`

    传送到指定的收藏位置（通过 Id 标识）。

    可能的结果：
    - 成功：`已传送到：<{id}> ({X}, {Y}) {name}`
    - 失败：`输入格式错误，用法：tpl <id>`
    - 失败：`{id} 不是有效的编号，编号应为整数`
    - 失败：`{id} 不是有效的编号，编号应在 0-{Count - 1} 之间`

- `rm <id>`

    移除指定的收藏位置（通过 Id 标识）。

    可能的结果：
    - 成功：`已移除：<{id}> ({X}, {Y}) {name}`
    - 失败：`输入格式错误，用法：rm <id>`
    - 失败：`{id} 不是有效的编号，编号应为整数`
    - 失败：`{id} 不是有效的编号，编号应在 0-{Count - 1} 之间`
